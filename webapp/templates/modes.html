Why These Modes?

Fix, Explain, Handoff — each with a clear job.

Iterate Card (Fix)

One atomic fix: Diagnosis → Fix Rules → Prompt Patch → Validation. Deterministic and copy‑pasteable with clear placement guidance.

Plan (Explain)

Align stakeholders: Context → Objective → Flow → Reasoning Path → Output. Tells the causal story for the change.

Ticket (Handoff)

Shareable brief: client/problem/test/goal/success. Lets PM/Ops push work without back‑and‑forth.

When To Use Which

  • Today’s fix: Iterate Card
  • Stakeholder alignment: Plan
  • Backlog handoff: Ticket

All three are deterministic and complementary. Start with Iterate; add Plan or Ticket when sharing.
